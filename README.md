SERP Snippet Tool
=================

[![Build Status][travis-image]][travis-url]
[![Dependency Status][gemnasium-image]][gemnasium-url]

Optimize the content of your titles and snippets to maximize your click-through
rate (CTR) from Google.

Setup
-----

```bash
$ bower install
$ npm install
```

Test
----

```bash
$ npm test
```

Build
-----

```bash
$ grunt build
```
